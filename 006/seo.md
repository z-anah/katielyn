## 1) Technical SEO (Shopify Foundation)

**Easiest**

* Verify site in Google Search Console
* Submit Shopify sitemap (`/sitemap.xml`)
* Connect Google Analytics
* Ensure HTTPS enabled
* Enable Shopify automatic canonical tags

**Medium**

* Fix crawl errors in Google Search Console
* Improve site speed (compress images, remove heavy apps)
* Mobile usability optimization
* Add structured data (Product, Breadcrumb, Organization)
* Optimize robots.txt

**Hard**

* Core Web Vitals optimization
* Advanced schema implementation
* Server response optimization
* App conflict cleanup
* Technical SEO audit with crawling tools
